# DoNotOpenGL

I started learning OpenGL using this [tutorial](https://learnopengl.com/Introduction) to render a music video for a song I'm currently writing. I decided to call the project DoNotOpenGL (lol).

The engine has evolved into a small scene-based rendering framework with editor support, runtime UI rendering, resource management, and a unified rendering pipeline. The lighting implementation is still in progress.

There is a demo scene called `Scene1` in `src/scenes/scene_1.cpp` you can run using the steps below. The textures are from [here](https://www.solarsystemscope.com/textures/).

---

## Repository Layout

```
CMakeLists.txt       # CMake build configuration (EngineCore library + App executable)
include/             # Public headers (engine, graphics types, scenes)
src/                 # Implementation files (engine, graphics helpers, GLAD, scenes)
shaders/             # Vertex and fragment shaders (copied to build output at build time)
assets/              # Textures and other runtime assets (copied to build output at build time)
```

> The `build/` directory is generated by CMake — do not edit it manually.

## Dependencies Setup

This project requires four external dependencies that are **not bundled** in the repository. Set each one up before building.

### GLAD (OpenGL Loader)

1. Go to the [GLAD generator](https://gen.glad.sh/) and use these settings:
   - Language: `C/C++`, Specification: `OpenGL`, API: `OpenGL 3.3` (or your version), Profile: `Core`, Generate a loader: **enabled**
2. Download and place the files:

```
src/glad/glad.c
include/glad/glad.h
```

### stb_image (Texture Loading)

1. Download [`stb_image.h`](https://github.com/nothings/stb/blob/master/stb_image.h)
2. Place it at:

```
include/stb/stb_image.h
```

### Dear ImGui (Runtime UI)

Clone or download [Dear ImGui](https://github.com/ocornut/imgui) and copy the files below.

Source files → `src/imgui/`:
```
imgui.cpp  imgui_draw.cpp  imgui_tables.cpp  imgui_widgets.cpp
imgui_impl_glfw.cpp  imgui_impl_opengl3.cpp
```

Header files → `include/imgui/`:
```
imgui.h  imgui_internal.h  imgui_impl_glfw.h  imgui_impl_opengl3.h
imgui_impl_opengl3_loader.h  imconfig.h  imstb_rectpack.h
imstb_textedit.h  imstb_truetype.h
```

### Miniaudio (Audio Playback)

1. Download [`miniaudio.h`](https://miniaud.io/) and place it at `include/miniaudio/miniaudio.h`
2. Create `src/miniaudio/miniaudio.cpp` with:

```cpp
#define MINIAUDIO_IMPLEMENTATION
#include "miniaudio/miniaudio.h"
```

## Creating a Scene

Scenes inherit from `DoNotOpenGL::Scene` and must implement:

| Method | Purpose |
|--------|---------|
| `init()` | Load resources, create objects |
| `reset()` | Reset scene state (called on timeline scrub) |
| `update(float deltaTime)` | Per-frame simulation logic |
| `render(Camera&, float aspectRatio)` | Draw the scene |
| `renderUI()` | ImGui panels (called in PlayGround mode) |
| `processInput(Input&, float deltaTime)` | Keyboard/mouse handling |
| `getCamera()` | Return the scene camera |

### Minimal Scene Skeleton

```cpp
#pragma once
#include "scenes/scene.h"

class YourScene : public DoNotOpenGL::Scene {
public:
    YourScene(DoNotOpenGL::Application *app) : Scene(app) {}

    void init() override
    {
        auto shader  = application->getResourceManager()->loadShader(
            "basic", "shaders/vertex/basic.vert", "shaders/fragment/basic.frag");
        auto texture = application->getResourceManager()->loadTexture(
            "earth", "assets/textures/earth.jpg");

        DoNotOpenGL::Object obj;
        obj.transform.position = glm::vec3(0.0f);
        obj.transform.scale    = glm::vec3(1.0f);
        obj.material.shader    = shader;
        obj.material.texture   = texture;
        objects.push_back(obj);
    }

    void reset() override { timeElapsed = 0.0f; }

    void update(float deltaTime) override { timeElapsed += deltaTime * speed; }

    void render(DoNotOpenGL::Camera &camera, float aspectRatio) override { /* draw */ }

    void renderUI() override { Scene::renderUI(); /* custom panels */ }

    void processInput(DoNotOpenGL::Input &input, float deltaTime) override {}

    DoNotOpenGL::Camera &getCamera() override { return camera; }

private:
    DoNotOpenGL::Camera camera;
    DoNotOpenGL::Light  mainLight;
    std::vector<DoNotOpenGL::Object> objects;
};
```

### Registering Your Scene

1. Add the header in `include/scenes/` and the implementation in `src/scenes/`.
2. Register the `.cpp` file in `CMakeLists.txt` under the `App` target.
3. In `src/main.cpp`, swap in your scene:

```cpp
app.setScene(std::make_unique<DoNotOpenGL::YourScene>(&app));
```

### Built-in Scene Utilities

The base `Scene` class provides ready-made ImGui helpers:

```cpp
renderLightUI(Light &light, int index);
renderCameraUI(Camera &camera);
renderObjectUI(Object &object, const std::string &category, int index);
renderTransformUI(Transform &transform, const std::string &prefix);
renderMaterialUI(Material &material, const std::string &prefix);
```

Predefined object collections (`lightObjects`, `materialObjects`, `skyObjects`, `rayObjects`) are also available for grouped rendering and UI organization.

## Runtime Notes

- Shaders must live under `shaders/vertex/` and `shaders/fragment/`. Asset paths are relative to the executable because CMake copies both directories into the build output.
- `EngineCore` is built as a static library; `App` links against it.
- Procedural mesh helpers are available via `Mesh::createTerrain(...)` and `Mesh::createCone(...)`.
